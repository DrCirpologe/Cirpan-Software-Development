GitHub Copilot mit Xcode verbinden

Schritt-für-Schritt-Anleitung zur Integration von GitHub Copilot in Xcode — für KI-gestützte Code-Vervollständigung direkt in deiner iOS & macOS Entwicklungsumgebung.

Voraussetzungen

Stelle sicher, dass folgende Punkte erfüllt sind, bevor du mit der Einrichtung beginnst.

🐙

GitHub-Konto

Ein kostenloses oder kostenpflichtiges GitHub-Konto unter github.com.

✨

Copilot-Abonnement

Ein aktives GitHub Copilot Abonnement (Individual, Business oder das kostenlose Kontingent).

🛠️

Xcode

Xcode 8 oder neuer, installiert aus dem Mac App Store auf einem Mac mit macOS 12 Monterey oder neuer.

🍎

macOS Monterey+

Dein Mac muss mindestens macOS 12 Monterey verwenden, um die Xcode-Erweiterung auszuführen.

Schritt-für-Schritt-Anleitung

Folge diesen Schritten, um GitHub Copilot in Xcode einzurichten.

1

GitHub Copilot for Xcode installieren

Öffne den Mac App Store auf deinem Mac und suche nach „GitHub Copilot for Xcode". Klicke auf Laden, um die offizielle App von GitHub zu installieren. Sie enthält die Xcode Source Editor Extension sowie den Copilot-Agenten, der im Hintergrund läuft.

2

Xcode Source Editor Extension aktivieren

Öffne die Systemeinstellungen (macOS Ventura/Sonoma: Systemeinstellungen → Datenschutz & Sicherheit → Erweiterungen, ältere macOS-Versionen: Systemeinstellungen → Erweiterungen). Wähle den Abschnitt Xcode Source Editor und aktiviere den Eintrag GitHub Copilot durch Setzen des Häkchens.

3

Mit GitHub anmelden

Starte die GitHub Copilot for Xcode App aus dem Launchpad oder Finder. Klicke in der App auf „Sign in to GitHub". Du wirst zu deinem Browser weitergeleitet, um dich mit deinem GitHub-Konto zu authentifizieren und die notwendigen Berechtigungen zu erteilen. Bestätige den Zugriff, kehre dann zur App zurück und schließe den Anmeldevorgang ab.

4

Copilot-Agent als Hintergrunddienst erlauben

Die GitHub Copilot App startet einen lokalen Agenten-Prozess, der die Kommunikation mit den GitHub-Servern übernimmt. macOS wird dich fragen, ob du den Dienst zulassen möchtest — klicke auf Erlauben. Du kannst den automatischen Start beim Login in den Einstellungen der Copilot App aktivieren, damit der Agent nach einem Neustart automatisch verfügbar ist.

5

Xcode neu starten

Starte Xcode vollständig neu (beende Xcode und öffne es erneut), damit die Erweiterung geladen wird. Öffne danach ein bestehendes oder neues Projekt und navigiere zu Editor → GitHub Copilot in der Menüleiste. Wenn der Eintrag erscheint, ist die Extension erfolgreich geladen.

6

Copilot in Xcode verwenden

Schreibe Code in einer Swift-, Objective-C- oder anderen unterstützten Datei. Copilot zeigt automatisch Vorschläge als grauen Geistertext an. Drücke Tab, um einen Vorschlag anzunehmen. Mit Esc lehnst du ihn ab. Über Editor → GitHub Copilot → Next Suggestion (oder das zugewiesene Tastenkürzel) kannst du durch weitere Vorschläge blättern.

💡 Tastenkürzel einrichten

Für ein flüssigeres Arbeiten empfiehlt es sich, individuelle Tastenkürzel für Copilot-Aktionen zu vergeben. Öffne dazu in Xcode Xcode → Einstellungen → Key Bindings und suche nach „Copilot". Du kannst Vorschläge annehmen, ablehnen und zum nächsten/vorherigen wechseln — ganz ohne die Maus zu benutzen.
